# 核心需求规范（金标准）

> 所有代码实现必须逐条满足。若需偏离，先改本文件，再重新生成代码。

- **REQ-1**：支持截图和录屏。
- **REQ-2**：录屏能自定义输出格式和质量。
- **REQ-3**：截图能单独复制到剪贴板且不落盘。
- **REQ-4**：截图能提供通过快捷键选择保存路径并保存。
- **REQ-5**：支持批注：矩形、椭圆、线条、箭头、铅笔、马克笔、文字、马赛克、高斯模糊、橡皮擦，并指出撤销、重做。
- **REQ-6**：选区支持：自动检测界面元素区域、像素级的鼠标移动控制、截图范围控制、取色器、
- **REQ-7**：支持多屏和高分屏。
- **REQ-8**：支持把图片作为窗口置顶显示，功能参考 snipaste。
